# Dotfiles

Personal dotfiles for macOS, managed using **GNU Stow**.

This repository contains configuration for:
- Zsh
- Ghostty terminal

More tools will be added incrementally.

---

## Structure

Each top-level directory is a **stow package**:

```text
dotfiles/
├── zsh/
│   └── .zshrc
├── ghostty/
│   └── .config/ghostty/config
